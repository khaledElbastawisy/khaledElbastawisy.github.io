# Personal Website

Static personal website for Khaled El-Bastawisy. The site is dependency-free and can be deployed on any static host.

## Files

- `index.html`: page structure and content
- `styles.css`: layout, theme, and responsive styling
- `script.js`: mobile nav, section highlighting, and reveal animations
- `assets/resume-industrial.pdf`: downloadable resume

## Preview locally

Run:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Publish

This project works well on GitHub Pages, Netlify, Vercel, or Cloudflare Pages because it only needs static file hosting.
